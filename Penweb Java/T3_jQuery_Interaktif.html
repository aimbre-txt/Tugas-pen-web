FAQ Interaktif

Apa itu jQuery? ▶
jQuery adalah pustaka JavaScript yang cepat, kecil, dan kaya fitur untuk manipulasi DOM, event, animasi, dan Ajax.
Bagaimana cara menggunakan .click() di jQuery? ▶
Metode .click() digunakan untuk menangani event klik pada elemen HTML.
Apa fungsi dari .toggle()? ▶
.toggle() berfungsi untuk menampilkan dan menyembunyikan elemen secara bergantian.
Apa perbedaan .hover() dan CSS :hover? ▶
.hover() bisa menjalankan JavaScript, sedangkan :hover hanya untuk styling CSS.
Apakah jQuery masih relevan? ▶
jQuery masih relevan untuk website sederhana dan manipulasi DOM cepat.
